# netscanner


Network scanning tool with features like:

- List Interfaces
- WiFi scanning
- WiFi signal strength (with chart)
- Ping CIDR with hostname, oui & mac address
- packetdump (TCP, UDP, ICMP, ARP)

must be run with sudo priviliges

## Install
```
cargo install netscanner
```
